# FutureTodo 要件定義

## 1. 目的

FutureTodoは過去のTodo履歴を分析し、
未来に発生する可能性が高いTodoを予測・提案する
Excel VBAアプリケーションである。

## 2. 開発方針

- Excel VBA
- オフライン動作
- UserFormを使用しない
- 長期保守可能な設計
- GitHubによるバージョン管理

## 3. 基本機能

### Todo管理

- Todo登録
- Todo編集
- Todo完了
- Todo履歴保存

### 履歴分析

- 過去完了データ保存
- 発生周期分析
- 曜日傾向分析

### 予測

- 次回Todo候補生成
- 予定日推定
- 信頼度表示

## 4. 将来的な機能

- Outlook CSV取込
- ルールエンジン
- 類似Todo検索
- 予測精度向上
